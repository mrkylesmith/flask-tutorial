# flask-tutorial
Following this [Flask Tutorial](https://www.youtube.com/playlist?list=PL-osiE80TeTs4UjLw5MM6OjgkjFeUxCYH)

# Install Flask
```
pip3 install flask
```

# Run App
```
export FLASK_APP=flaskblog.py
flask run
```

# Or, we can put this into the flaskblog.py
```
if __name__ == '__main__':
	app.run(debug=True)
```
